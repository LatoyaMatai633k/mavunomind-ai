# MavunoMind AI – Stage 2 Prototype

MavunoMind AI is a web-based smart farming dashboard prototype built for the Beorchid Africa Developers Hackathon 2026 (Stage 2).

## Overview
This prototype demonstrates how African farmers can:
- Ask an AI assistant about crop or livestock issues
- Receive simple guidance and prevention tips
- View disease alerts
- Access farming knowledge resources
- See multilingual expansion plans

## Prototype Scope
This is a frontend prototype demonstrating user experience and system flow. The AI responses are simulated for demo purposes.
